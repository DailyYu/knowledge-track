# knowledge-track 知识复习追踪

## 简介

### 前端
目录：knowledge-track-backend
技术栈：spring-boot、 mysql、 mybatis


### 后端
目录： knowledge-track-frontend
技术栈：React18、vite、Ant Design

## 如何启动



### 前端

```bash
npm install
npm run dev
```

### 后端

1、安装Maven和MySQL数据库
2、执行ddl.sql脚本创建数据库和表
3、更改application.properties中数据库配置，连接自己的数据
